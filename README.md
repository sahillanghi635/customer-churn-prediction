# Customer Churn Prediction

## 📌 Project Overview

Customer churn prediction helps businesses identify customers who are likely to stop using their services.
This project builds machine learning models to predict customer churn using a telecom dataset.

Predicting churn helps companies improve **customer retention, marketing strategies, and service quality**.

---

## 📂 Dataset

Dataset used: **Telco Customer Churn Dataset**

Features include:

* Customer demographics
* Account information
* Service usage
* Billing information
* Customer churn status

Target variable:

* **Churn** (Yes / No)
